# System-Tracking-PWA
Progressive web app for tracking front for dissociative/plural systems

F-Spec (Rough Draft)

**Personal**

*Members*

Need:

Member profile (Name, pronouns, description, tags, colour, profile picture, custom text fields, privacy settings, fronting history)
Custom front (Name, description, tags, colour, profile picture, privacy settings, fronting history. Not a member and stored in a separate tab)
Ability to set one/none/multiple members and/or custom fronts as front and remove them from front
Field indicating how long each member has been in front 
Ability to put members in folders and archive them
Ability to add a text note to each member in front (visible by other users)
Ability to add a status note to each member in front (visible by other users)
Ability to add private comments on fronting members (not visible by other users)
Storage of fronting history, including note and comments and start/end time
Ability to edit fronting history (retroactively add a front for a time period or start/end time, delete one, edit the note or add a comment, edit who the fronter was) 

Nice-to-have:
Profile picture history/storage page (maybe more trouble than worth?)
Journal feature and private chat feature (where you can pick what part is putting the message)
Leaving a message for the other user/other user's parts? (would show up in a "messages" tab on that member's profile maybe?) 
More customization on fields / stuff that is visible to the other user when logging fronts 

Really want but would survive without: Ability to import data from existing app. It has an export file that's a json with all the history and members in it

*Social*

Need:
Ability to see list of members and their profiles of other users (that have been added as friends) 
Ability to see who the other user has logged in front and the notes on who they've put in front 
Ability to get a push notification when the other user has changed who is in front 
(privacy settings used to hide members or fields, or prevent sending a notification when fronting)

*Front history*

Need:
Some sort of graphical rendering of the data on a timeline, like how the old app does it.

*Journal*
Nice-to-have: 
Ability to make private journal entries with some formatting and selecting what member is writing it if wanted/ colours. 
Format options including by date and in a long list (like a discord channel, scroll through newest to oldest)
Search function (just exact/partial matches)
